题目：

  给一个数组，里面如果有重复的元素，那么只留一个，将其他重复的元素删除，比如[1,2,3,4,2,3,3]，结果是[1,2,3,4]。

方法一：

  1. 构建一个新的数组存放结果；
  2. for循环中每次从原数组中取出一个元素，用这个元素循环与结果数组对比；
  3. 若结果数组中没有该元素，则存到结果数组中。

总结：

  以上三种方法都能用，第一种方法的时间复杂度达到了O(n2)，空间复杂度为O(1)，效率较低，用的时候还是推荐方法二和方法三，方法二的时间复杂度是排序算法的复杂度再加上O(n)，空间复杂度为O(1)；而方法三的时间复杂度是O(n)，空间复杂度为O(n)，如果不计较空间复杂度的话建议还是用方法三，如果不允许使用额外空间的话还是方法二好用一些。
