方法四

  css3的flex布局也是一个好方法，将display属性设置为flex，之后容器就被称为flex容器，这种容器默认有两条轴，一条是水平的主轴，另一条是垂直的交叉轴。flex-direction属性就是用来指定主轴的方向或者叫做容器里面项目的排列方向，它默认为row，这个时候主轴是水平的；设置为column的时候，主轴就变成垂直的了。当它为row的时候，要做到水平居中，就要设置一个属性：justify-content，这个属性设置为center的时候，容器里面的项目就会在主轴上居中；当它不为row的时候，就要设置另外一个属性：align-items或者align-content，将他们设置为center，就能够将容器里的项目沿着交叉轴居中对齐。代码如下：
